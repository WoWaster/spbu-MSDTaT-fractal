# Генератор фракталов на C

![Статус сборки](https://github.com/WoWaster/spbu-MSDTaT-fractal/actions/workflows/build.yml/badge.svg)

Простой генератор фракталов на C с сохранением в не менее простой формат изображений PPM (P6).

Поддерживает 4 вида фракталов:
1. Множество Мандельброта
2. Множество Жюлиа
3. Треугольник Серпинского
4. Горящий корабль

[!TIP]
Кстати, в артефактах сборки можно найти готовые изображения.

## Сборка и запуск

Тестировался только на Linux, могут быть проблемы под Windows.

Потребуется компилятор C (gcc или clang), а также CMake для сборки.

Далее выполните
```console
$ cmake -B ./build -DCMAKE_BUILD_TYPE=Release
$ cmake --build ./build
```

Затем запустите
```console
$ ./build/fractal_gen --type mandelbrot
```
для генерации множества Мандельброта.

Другие поддерживаемые типы: `julia`, `sierpinski`, `burningShip`.

Также поддерживается указание размеров изображения и названия выходного файла (по умолчанию изображение 800x600, файл `output.ppm`).
Об их использовании можно узнать запустив программу с флагом `--help`.
